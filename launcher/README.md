# LovelyMem 启动器

这是一个为 LovelyMem 应用程序设计的无黑框启动器，使用 Go 语言开发。

## 功能特点

- **无黑框启动**：启动 Python 应用程序时不会显示命令行窗口
- **自动检测项目目录**：即使启动器位于不同位置，也能自动找到主项目目录
- **自动查找 Python 解释器**：自动在虚拟环境中查找 Python 解释器
- **应用图标支持**：使用项目的 logo.ico 作为应用程序图标
- **错误处理**：提供友好的错误提示，便于用户排查问题

## 使用方法

直接双击 `LovelyMem.exe` 即可启动应用程序。启动器会：

1. 自动查找项目目录和 `.venv` 虚拟环境
2. 使用找到的 Python 解释器运行 main.py
3. 完全隐藏命令行窗口，只显示应用程序界面

## 编译方法

如需重新编译启动器，请按照以下步骤操作：

1. 确保已安装 Go 语言环境
2. 在 launcher 目录中运行以下命令：

```bash
# 编译带图标的最小化可执行文件
go build -ldflags "-H windowsgui -s -w" -trimpath -o ../LovelyMem.exe
```

## 技术说明

- 使用 `syscall.SysProcAttr.HideWindow = true` 隐藏命令行窗口
- 使用 `rsrc` 工具添加应用程序图标
- 使用 `-ldflags "-H windowsgui -s -w"` 和 `-trimpath` 优化可执行文件大小

## 注意事项

- 启动器需要能够访问项目的 `.venv` 虚拟环境
- 如果遇到错误，启动器会显示错误信息并等待用户按键后退出
- 启动器会自动查找项目目录，但建议将启动器放在项目根目录或其子目录中
